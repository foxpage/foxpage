# `foxpage-admin`

`foxpage-admin` is the front-end project for `FoxPage`.

### front-end stack

```txt
react + redux + saga + immer + styled-components + Webpack
```

### UI

```txt
Ant Design
```

### start

```txt
npm start
```

### build

```txt
npm run build
```
